CRIANDO UM NOVO PROJETO REACT (PASSOS)

1- Criar o projeto com o comando
  npx create-react-app nome-projeto
  caso não dê certo, rodar o comando npm i -g npm e tentar novamente

2- Limpar a base do projeto
  public: remover os 5 arquivos desnecessários e deixar somente o 
  index.html

  src: remover os 6 arquivos desnecessários e deixar somente App.js 
  e index.js

Daqui para baixo, construir de acordo com a sua necessidade.


Como criar uma pg CRUD no react
 
 1- Criar uma pagina vazia

 2- Criar o serviço desse CRUD

 3- Criar a pagina inicial do CRUD (no caso a tabela e o botão para iniciar)

 4 - Criar o modelo da nossa entidade que representa a regra de negocio
